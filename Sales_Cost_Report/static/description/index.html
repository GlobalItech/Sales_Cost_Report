Sales Cost Report

This reporting app allows its users to carry out the periodic sales and cost analysis in seconds. This report is an easy tool for getting various type of information. Benefits of this report include:
  • Ease of access is the one of the main features of this report. As you can see, this report can be initiated from the very front menu in Sales’ module.
  • Simple User interface is everybody’s need in this complex world of technology. So we made it super simple for its users to use.
  • A report can be obtained of flexible dates. There is no binding on users to select a week, monthly or a year period.
  • A specific category can be selected plus all categories can also be selected for the report
  • In case of Multi Company, report can be obtained for each company. Once products are selected for a specific dates, expost xlsx report.
  • Once report is downloaded in local memory, open it and you are good to go with the report. Analysis can be made with the available information/excel report which includes: a. Date b. Customer c. Code d. Product Name e. Product Quantity f. Cost g. Total cost h. Sale Value i. Net Profit
